## Tips for Recording

1. **Keep it fast-paced** - 2 minutes is tight
2. **Show, don't tell** - Visual demonstrations > explanations
3. **Highlight prize eligibility** - Make it clear for judges
4. **Smooth transitions** - Use screen recordings, not static screenshots
5. **Clear audio** - Good microphone quality
6. **Practice** - Rehearse timing before recording

---

## Backup Plan (If Live Demos Fail)

1. Use pre-recorded screen recordings
2. Show screenshots with annotations
3. Use animated transitions between sections
4. Add text overlays for key points
