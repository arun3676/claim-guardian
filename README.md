# ClaimGuardian AI

Medical billing and claims management platform powered by AI.

## Structure

- `frontend/` - Next.js application with Vercel AI SDK
- `backend/` - Optional Python backend services
- `kestra-flows/` - Kestra workflow definitions for automation
- `oumi-training/` - Model training configurations
- `mcp-servers/` - Custom Cline MCP servers for medical billing
- `docs/` - Documentation and guides

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Kestra Flows

Deploy workflows to Kestra server for automated claim processing.
